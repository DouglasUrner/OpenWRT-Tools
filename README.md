# OpenWRT Tools

Scripts for configuring OpenWRT devices.

To use, copy the desired scripts to the OpenWRT machine with ```scp```:

```bash
scp -O [the scripts you want] root@[router IP]:/tmp
ssh root@[router IP]
cd /tmp
```
Then, run the scripts you want.
